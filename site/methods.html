Methods

How Pathogen Watch turns NCBI's nightly snapshot into a daily surveillance feed.

Data source

Each morning at 09:00 UTC (04:00 EST / 05:00 EDT), Pathogen Watch reads the latest NCBI Pathogen Detection metadata snapshot for four taxgroups: Salmonella, Escherichia coli & Shigella (filtered to STEC by Shiga-toxin gene presence), Listeria, and Campylobacter. The snapshots are published nightly at ftp.ncbi.nlm.nih.gov/pathogen/Results/ under release accessions of the form PDG000000002.NNNN.

Cluster definition

NCBI's PDS (Pathogen Detection SNP) clusters are recomputed nightly using a 50-SNP single-linkage threshold within each reference target. We accept those cluster assignments as the unit of analysis. For each isolate we record its PDS_acc (cluster membership), epi_type (clinical = human; everything else = nonhuman), min_same (SNPs to the nearest other isolate in the same cluster), and min_diff (SNPs to the nearest isolate in any other cluster).

Alert triggers

Three day-over-day events generate alerts:

New human cluster. A cluster appears in today's snapshot that was not in yesterday's, and it contains at least 2 human clinical isolates.

Human added to existing cluster. A previously-tracked cluster has a new human PDT_acc today.

Nonhuman added to human cluster. A cluster with at least one human isolate gains a new nonhuman isolate today.

Near-neighbor surveillance

Independently of clustering, we flag any nonhuman isolate whose min_diff is ≤5, ≤10, or ≤50 SNPs. These are isolates whose nearest neighbor in a different cluster is tight enough that the next PDS recomputation may merge the two clusters. When the neighboring cluster contains humans, this is a leading indicator.

Important caveats

min_same is an upper bound, not the exact human↔X distance. If a human isolate's min_same = 3, its nearest neighbor within the cluster is 3 SNPs away — but that neighbor might be another human. For exact pairwise distances between specific isolates, retrieve the per-cluster SNP matrix from NCBI directly via the link on each cluster page.

STEC filtering uses AMRFinderPlus stx hits. Stx-negative pathogenic E. coli (EPEC, ETEC, EHEC variants without typeable stx) will not appear under the STEC label.

NCBI is the source of truth. Pathogen Watch is a visualization and diffing layer; the clustering and SNP calls are NCBI's. If a cluster you care about disappears here, it has been split, merged, or reassigned upstream — check the live browser.

Reproducibility

Every daily build is committed to github.com/bhaley1/pathogen-watch. The per-pathogen state snapshots are tracked in state/ and the rendered digests in digests/; you can git log any cluster to see exactly when it was first observed and how it has changed.

About

Pathogen Watch is an open-access daily surveillance digest of human–animal–environmental SNP clusters in foodborne pathogens, derived from the NCBI Pathogen Detection nightly snapshot. Built by Bradd Haley, PhD at the USDA Agricultural Research Service.
