Schemas

clusters.json — one record per active human-containing cluster:

{
  "pathogen": "Salmonella",
  "pds_acc": "PDS000123456.7",
  "n_human": 4,
  "n_nonhuman": 2,
  "tightest_human_min_same": 3,
  "latest_collection": "2026-05-14",
  "countries": ["USA"],
  "nonhuman_sources": ["bovine - milk", "poultry"],
  "ncbi_url": "https://www.ncbi.nlm.nih.gov/pathogens/isolates/#PDS000123456.7",
  "site_url": "https://bhaley1.github.io/pathogen-watch/clusters/salmonella-pds000123456-7.html"
}

events.json — one record per alert event:

{
  "detected_date": "2026-05-21",
  "event_type": "new_human_in_cluster",
  "pathogen": "Salmonella",
  "pds_acc": "PDS000123456.7",
  "summary": "Cluster PDS000123456.7 gained 1 new human case(s) (now 4 total)",
  "new_pdt_accs": ["PDT001234567.1"]
}

About

Pathogen Watch is an open-access daily surveillance digest of human–animal–environmental SNP clusters in foodborne pathogens, derived from the NCBI Pathogen Detection nightly snapshot. Built by Bradd Haley, PhD at the USDA Agricultural Research Service.

Source code: github.com/bhaley1/pathogen-watch
