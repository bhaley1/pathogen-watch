Data & API

All Pathogen Watch data is open access. Use it, build on it, cite it.

Downloads

Refreshed every morning at 09:00 UTC. Same data the site is built from.

clusters.json clusters.csv events.json events.csv near_neighbors.json near_neighbors.csv

RSS / Atom

Subscribe to feed.xml in any RSS reader (Feedly, Inoreader, NetNewsWire, Thunderbird) for a notification each time a new alert event is detected.

Schemas

clusters.json — one record per active human-containing cluster:

{
  "pathogen": "Salmonella",
  "pds_acc": "PDS000123456.7",
  "n_human": 4,
  "n_nonhuman": 2,
  "tightest_human_min_same": 3,
  "latest_collection": "2026-05-14",
  "countries": ["USA"],
  "nonhuman_sources": ["bovine - milk", "poultry"],
  "ncbi_url": "https://www.ncbi.nlm.nih.gov/pathogens/isolates/#PDS000123456.7",
  "site_url": "{{ site.base_url }}/clusters/salmonella-pds000123456-7.html"
}

events.json — one record per alert event:

{
  "detected_date": "2026-05-21",
  "event_type": "new_human_in_cluster",
  "pathogen": "Salmonella",
  "pds_acc": "PDS000123456.7",
  "summary": "Cluster PDS000123456.7 gained 1 new human case(s) (now 4 total)",
  "new_pdt_accs": ["PDT001234567.1"]
}
